# 대출 이자 계산기 Blueprint

## 1. 개요

본 프로젝트는 사용자가 대출 원금, 기간, 이자율 등 기본 정보를 입력하면 다양한 상환 방식에 따른 총 이자 및 월별 상환 스케줄을 계산하고 비교해주는 웹 애플리케이션입니다. 사용자는 이를 통해 자신에게 가장 유리한 대출 상환 계획을 수립할 수 있습니다.

## 2. 핵심 기능 및 디자인

### v1.0 (현재 버전)

*   **상환 방식 동시 계산 및 비교:**
    *   사용자가 대출 금액, 기간, 이자율을 입력하면 '원리금균등', '원금균등', '만기일시', '체증식' 네 가지 상환 방식의 총 이자를 모두 계산하여 한눈에 비교할 수 있도록 카드 형태로 보여줍니다.
*   **상세 상환 스케줄 테이블:**
    *   각 상환 방식 카드 아래에 있는 버튼을 클릭하면, 해당 방식의 상세 상환 내역(회차, 상환원금, 이자액, 납부액, 잔액)이 테이블 형태로 표시됩니다.
*   **부가 기능:**
    *   **거치 기간:** 거치 기간을 설정하여 이자만 납부하는 기간을 계산에 포함할 수 있습니다.
    *   **중도 상환/금리 변동:** 특정 회차에 추가 상환을 하거나 금리가 변경되는 시나리오를 시뮬레이션할 수 있습니다.
*   **디자인:**
    *   깨끗하고 현대적인 UI를 적용하여 사용자가 쉽게 정보를 입력하고 결과를 이해할 수 있도록 합니다.
    *   모바일 반응형 디자인으로 다양한 기기에서 최적의 경험을 제공합니다.

## 3. 이번 변경 요청 계획

*   **기능 변경:**
    1.  기존 상단 탭 형태의 상환 방식 선택 버튼(`원리금균등`, `원금균등` 등)을 제거합니다.
    2.  서버 액션을 수정하여, '대출 이자 계산' 버튼 클릭 시 네 가지 상환 방식 모두에 대한 총 이자와 전체 상환 스케줄을 한 번에 계산하도록 변경합니다.
    3.  계산 결과 섹션을 새롭게 디자인하여, 각 상환 방식별 총 이자를 요약하여 보여줍니다.
    4.  결과 요약 아래에 각 상환 방식의 이름이 적힌 버튼을 추가합니다.
    5.  이 버튼 중 하나를 클릭하면, 해당 상환 방식의 상세 상환 스케줄(회차, 상환원금, 이자액, 납부액, 잔액)이 담긴 테이블이 나타나도록 UI를 구현합니다.

*   **작업 순서:**
    1.  `blueprint.md` 파일 생성 (완료)
    2.  `app/lib/loanLogic.ts` 파일 수정: 네 가지 상환 방식을 모두 계산하는 새로운 서버 액션 `calculateAllLoanScenarios`를 구현합니다.
    3.  `app/components/LoanPlanner.tsx` 파일 수정:
        *   기존 상환 방식 선택 버튼 제거
        *   새로운 서버 액션을 사용하도록 `useActionState` 업데이트
        *   네 가지 방식의 계산 결과를 모두 표시하는 새로운 UI 섹션 구현
        *   상세 상환 스케줄 테이블을 보여주는 기능 및 UI 구현
